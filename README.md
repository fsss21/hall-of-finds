# Hall of Finds
**0.07 Зал находок(каталог с индивидуальными разделами)**
Проект на React + JavaScript + React Router

## Установка

```bash
npm install
```

## Запуск проекта

Для запуска в режиме разработки:

```bash
npm run dev
```

Проект будет доступен по адресу `http://localhost:5173`

## Сборка

Для создания production сборки:

```bash
npm run build
```

## Предпросмотр production сборки

```bash
npm run preview
```
